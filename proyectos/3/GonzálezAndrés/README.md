## Requerimientos 
 
- Gestor de paquetes npm
- Correr los siguientes comandos dentro del folder fiunam-explorer:
``` bash
# instalar dependencias
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
